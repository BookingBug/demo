## Basic Usage
### Installation

To install required packages

`npm run i`

### Running The Projects
#### Public booking journey

##### Default

`npm run p`

##### Advanced

`npm run p-dev`

`npm run p-staging`

`npm run p-prod`

#### Studio journey

##### Default

`npm run s`

##### Advanced

`npm run s-dev`

`npm run s-staging`

`npm run s-prod`
